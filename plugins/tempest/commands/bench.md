Load and follow the `bench` skill to measure front-end performance and Core Web Vitals.

Page or change to measure: $ARGUMENTS

Work through the skill's procedure end to end: pick a production-like build and a budget or baseline,
capture vitals with agent-browser (the native tool on pi, the CLI on Claude Code), take multiple
samples and use the median, compare against the budget/baseline, and report concrete numbers in a
table — not impressions.
